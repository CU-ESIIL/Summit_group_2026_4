## User Inputs

- For each of the following inputs, succinctly prompt the user for their preferences.
- If a prompt is marked as **Mandatory**, the user must provide a response.
- If a prompt is marked as **Optional**, the user may choose to skip it or leave it blank.
- If the user provides input or a response that does not answer the given prompt, kindly re-ask the prompt.

### Required Inputs

- **Mandatory Input:** Topic or discipline defined by the user.
- **Mandatory Input:** Open access journal.
- **Optional Input:** Range of publication dates.
- **Optional Input:** Spatial scale or target region of interest (state, county, custom boundary, or bounding box).
- **Optional Input:** User-defined selection criteria (e.g., language, impact factor).

---

## Example Inputs Provided by a User

**Agent:**  
> "For what topic or discipline would you like to find data?"

**User:**  
> "I would like to perform a meta-analysis of the morphology of California's native wildflowers."

---

**Agent:**  
> "Which open access journal(s) would you like to search for available data?"

**User:**  
> "PLOS ONE, bioRxiv, BioMed Central"

---

**Agent:**  
> "Would you like to specify the range of publication dates?"

**User:**  
> "No"

---

**Agent:**  
> "Would you like to specify the spatial scope or extent?"

**User:**  
> "Yes, please limit studies that include data from the state of California."

---

**Agent:**  
> "Would you like to provide any additional criteria?"

**User:**  
> "Yes, please include all papers that are in English or Spanish."

---

Once all mandatory inputs are provided, and optional inputs are either provided or skipped, proceed with the following tasks.

---

# Task 1 - Determine Access to Journal(s)

For the journals specified by the user, determine the agent's ability to access and download associated data.

- If at least one journal is inaccessible and at least one other journal is accessible:
  - Pause the workflow.
  - Display a failure error message describing which journal(s) were inaccessible.

### Example

> "Nature is not accessible to the agent. PLOS ONE is accessible to the agent. Would you like to proceed with the journals specified earlier, or refine your input?"

- If no journals provided by the user are accessible:
  - Do not proceed without additional journal input from the user.

---

# Task 2 - Find Relevant Articles

For each journal determined to be accessible:

- Identify articles relevant to the provided topic.
- Ensure the articles conform to other provided user inputs, such as:
  - Spatial constraints
  - Temporal constraints
  - Additional selection criteria

---

# Task 3 - Find Relevant Datasets

For each journal article determined to be relevant and conforming to user inputs, identify whether the article contains any of the following:

- Raw data within the article itself
- References or links to raw data available in an online repository (such as Zenodo, Dryad, MorphoSource, etc.)
- Supplementary materials that contain data
- Figures or tables that contain data
- Tabular data presented in text

Record and save each qualifying article as a citation and DOI in:

```text
/workflows/STEP1/output/Step1Output.md
```
